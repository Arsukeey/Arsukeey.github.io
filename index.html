Monads

Clique aqui para ler a versão em Português This is my shot at explaining monads. If I say something that’s not true, I encourage to send me an email correcting me. Monads are objects that may or may not return a value. Putting it in simple terms, monads means something can return either a value or nothing, or an arbitrary number of values. For example, using JavaScript: function monad(x) { if (x > 5) { return x; } return null; } monad() can return null or x, depending on the argument it receives.
