Mônades

Click here to read the English version

Essa é a minha tentativa em explicar mônades. Se eu disse alguma coisa errada, eu peço que você me corrija por meio de um email.

À grosso modo,

Mônades são objetos que podem ou não retornar um valor.

Explicando bem por cima, um mônade quer dizer que algo pode retornar um valor ou um erro, ou nada, ou um número arbitrario de valores.

Um exemplo simples usando ECMAScript:

function monad(x) {
    if (x > 5) {
        return x;
    }
    return null;
}

monad() pode retornar null ou x, dependendo do argumento que ela recebe. PORÉM, você pode também retornar um erro dali, ou até mesmo uma lista com um tamanho n.
