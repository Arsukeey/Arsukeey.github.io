A Luta

Talvez você já tenha escrito alguma coisinha em C++ ou BASIC, mas tá na hora de botar a mão na massa e aprender como fazer as coisas para o mundo real, e do jeito certo.

Quando você está começando a programar, você pode ficar confuso com qual linguagem de programação você deve aprender (mesmo que você sequer tenha certeza do que elas se tratam).

Então eu vou te dar uma ajuda. Pare de tentar fazer isso!

Você está gastando tempo precioso que você poderia estar usando para estudar ou programar!

Basta escolher a linguagem que você acha que pode gostar. Se você não tem idéia de qual você deve começar, bem, então eu vou te dar algumas dicas.

Lembrete: Por enquanto você está procurando aprender a lógica de programação, não os detalhes da linguagem.

Linguagens de programação que os iniciantes podem usar para aprender

  1. Matemática. Sério, esta é a parte mais importante. Quando você está começando a aprender a lógica por trás da magia, a matemática será sua melhor aliada.

  2. Python. Linguagem simples e madura com uma sintaxe próxima ao inglês. É uma habilidade boa de se ter.

  3. C. Linguagem antiga mas muito madura. A maioria das linguagens modernas de programação tem suas raízes aqui, e se você acha que não são razões suficientes, com C você aprenderá sobre como o computador faz sua mágica.

  4. Scheme. As pessoas podem não concordar comigo sobre isso, mas este é meu blog. Na minha opinião, Scheme é bom de se aprender pois a sintaxe é quase inexistente. E! Você tem Structure and Interpretation of Computer Programs (SICP), que é um dos melhores livros de Ciência da Computação que existem, para te introduzir à Scheme.

Onde você pode aprender (recursos)

Eu pessoalmente aprendi programação usando o Sololearn (não estou sendo pago para falar isso) e acho que é uma plataforma incrível. Se você não gosta, você sempre pode ler um livro! Há muitos deles. E você não precisa pagar por eles, já que você tem zlib.

Alguns livros que recomendo são:

  • SICP (como eu disse antes)
  • The C Programming Language, 2ª Edição // Depois de terminar esse, leia Modern C
  • The Little Schemer
  • The Practice of Programming
  • Code Complete
  • Clean Code
  • The Pragmatic Programmer —

Agora, você precisará de um espaço de trabalho para aprender. Falando de hardware, qualquer coisa funciona. Mas tudo será mais fácil se você instalar uma distribuição Linux e um editor de texto/IDE. A maioria das pessoas gosta de usar o Sublime Text ou o VS Code para editar texto, mas você pode usar qualquer coisa que funcione para você. (Se você se importa, eu estou usando o neovim e planejo escrever sobre isso no futuro)

E depois?

Ok, agora você sabe o básico da programação. Boa! Meus sinceros parabéns.

Agora, aprenda sobre

  • HTML / CSS / JS. É a carne e o osso da web. Como a web é a melhor aplicação de GUI, você precisa aprender isso.

  • Concorrência. Aprenda como fazer programas assíncronos.

  • Git. Contribua e aprenda para com projetos de código aberto.

  • Java / Swift. Você usará o Java para criar aplicativos móveis Android e o Swift para aplicativos iOS.

  • Programação funcional. Aprenda Haskell, dialetos de Lisp e Erlang / Elixir. Sério, você será um programador melhor, você pensará mais matematicamente e, se aprender bem os conceitos, irá fazer aplicações mais seguras e facéis de entender.

  • C++. Este vai te ensinar Programação Orientada a Objetos, que ainda (infelizmente) é uma forte habilidade no mercado de trabalho atualmente. Depois que você estiver de bem com C++, aprenda Rust. “Rust é como 5 linguages combinadas, mas em vez de acabarem como 5 crianças em um casaco, ele fica tipo os power rangers” Rust tem todas as boas características do C++, mas sem a parte chata.

  • Redes.

  • Node.js, npm e React
