Getting started with Programming

The struggle

When you are getting started with programming, you may be confused with which programming language you should learn (even though you are not even sure what those are).

So I’ll give you some help. Stop trying to do that!

You are spending precious time you could be studying or programming!

Just pick whatever language you think you may like. If you have no idea of which one you should start with, I’ll give you some tips.

Reminder: Right now you are looking to learn programming logic, not the ins and outs of the language.

Programming languages I think begginers can use to learn

  1. Math. Seriously, this is the most important part. When you are starting to learn the logic behind the magic, math will be your best ally.

  2. Python
